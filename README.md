# 💸 Expense Tracker (Python CLI)

A simple and efficient **command-line Expense Tracker** built using Python.  
This project helps users record, view, and manage daily expenses using a CSV file as storage.

---

## 🚀 Features

- ➕ Add new expenses (Date, Category, Amount, Note)
- 📋 View all expenses in a clean table format
- 💾 Data stored locally in CSV file (no database needed)
- ⚡ Fast and lightweight CLI-based application
- ❌ Input validation for incorrect data

---

## 🛠️ Tech Stack

- Python
- Pandas
- Tabulate
- CSV (for data storage)

---
